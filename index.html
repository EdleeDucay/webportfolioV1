Projects

Book Tracker

Book Tracker is an android application that me and a team of five other people created. The purpose of the app is to allow users to share books with each other and track who the books have been lent to. We implemented the Firebase Cloud Firestore database to make real time updates. Google Maps API is implemented to set a location to receive books. The Google Books API lets users scan books that are in the Google Books database and add them to their collection.

Baby Names

The demo is a web-app version of the original project modified to use Flask, html, css bootstrap to display the information in a more user friendly/cleaner fasion. Baby Names is one of my first ever 'big' projects in my computer science career. It was designed to read in data from a text file that contains names sorted by ranking, first name, frequency, gender, birth-year. The data I was given is official data from the Province of Alberta recording names from January 1, 1980 to December 31, 2017. Proof that it's official is that I can find my own name and that I'm the only Edlee born in Alberta!
